## üöÄ How to Install and Run (.exe version)

Because this is a standalone executable file, Windows might flag it as "unrecognized." **Please follow these steps to run the application safely:**

1. **Download** the `BY-Downloader.exe` file from the [Releases](https://www.google.com/search?q=link-to-your-release-here) section.
2. **Unblock the file:**
* Right-click on the downloaded `.exe` file.
* Select **Properties**.
* At the bottom of the "General" tab, look for the **Security** section.
* Check the box that says **"Unlock"** (or "Unblock").
* Click **Apply** and then **OK**.


3. **Run as Administrator:** Right-click the file and select "Run as Administrator" for the best experience.

> **Note:** If Windows Defender shows a blue "Windows protected your PC" popup, click on **"More info"** and then click **"Run anyway"**.
